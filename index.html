Nessus Vulnerability Analysis

In the Nessus Vulnerability Analysis project, I concentrated on identifying and addressing security vulnerabilities on a metasploitable machine. Honeypot with Azure Sentinel

In the Azure Sentinel project, I concentrated on deploying and configuring a custom Azure Sentinel homelab to monitor and analyze security events. Specifically, I set up the homelab to capture and investigate failed Remote Desktop Protocol (RDP) attempts from a device intentionally exposed to potential attackers.

  • Hands-On Experience with Azure Sentinel: Gained practical experience in setting up and configuring Azure Sentinel as a Security Information and Event Management (SIEM) tool.
  • Log Analytics and Monitoring: Created and used PowerShell scripts to generate security logs for failed RDP attempts and established a custom log in Azure Log Analytics Workspace to capture and analyze this data.
  • Visualization and Reporting: Implemented a world map visualization to track failed RDP attempts and other security metrics, offering valuable insights into the global landscape of potential threats.
